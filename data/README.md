# Data Directory

This directory contains the SQLite database file for Telugu Tutor.

The `telugututor.db` file is automatically created when the server starts.

## Database Tables

- `users` - User accounts (email, phone, name)
- `user_profiles` - Learning progress and settings
- `grapheme_mastery` - Individual grapheme learning progress
- `practice_sessions` - History of practice sessions
- `sessions` - Authentication tokens
